# Notris

## Controls

| Key                                   | Action       |
| ------------------------------------- | ------------ |
| A, Left, Numpad4                      | Left         |
| D, Right, Numpad6                     | Right        |
| S, Down, Numpad5, Numpad2             | Soft Drop    |
| Space, Numpad9, NumpadAdd, ShiftRight | Hard Drop    |
| Up, E, Numpad8                        | Rotate Right |
| Z, W, /, Numpad8, NumpadDivide        | Rotate Left  |
| O                                     | Reset        |
| P, K                                  | Pause        |
